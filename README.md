# 📊 Social Media Analytics Portfolio

## Overview
Complete data analysis of 2,000 social media posts across 5 platforms (2024).

## Key Findings
- Best performing platform: analyzed in notebook
- Optimal posting time: identified via heatmap analysis
- Content type insights: Video & Reels outperform

## Project Structure
```
├── data/           → Dataset & generator script
├── notebooks/      → Main Jupyter analysis notebook
├── visualizations/ → 7 generated charts
├── src/            → Reusable functions
└── reports/        → PDF report (coming soon)
```

## Tech Stack
Python | Pandas | Matplotlib | Seaborn | Plotly | Jupyter

## How to Run
```bash
pip install -r requirements.txt
python data/generate_dataset.py
jupyter notebook notebooks/social_media_analysis.ipynb
```
